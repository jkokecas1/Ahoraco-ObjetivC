## Juego del Ahorcado

_El legendairo Jugeo del Ahorcado en la terminal de XCode utilizando las estructuras de datos de Objetiv-c NSString, NSMutablearray para la manipulacion de las cadenas de entrada.

## Comenzando 🚀
